# Flappy Pink (Juego web)

Un minijuego estilo "flappy" súper amable y en tonos rosados. No usa ninguna marca registrada: es un personaje genérico y adorable.

## Cómo jugar
- Abre `index.html` en tu navegador (doble clic).
- Toca o presiona **espacio / flecha arriba** para volar.
- Pasa entre las columnas y junta estrellas para sumar puntos.
- **Modo amable** (activado por defecto): si chocas, te ayudamos a seguir.
- **Modo normal**: desactívalo con el botón si quieres más reto.

## Archivos
- `index.html`: el juego.
- `assets/`: imágenes (fondo, personaje, corazones, estrellas, brillos, logo, favicon).

## Personalización rápida
- Cambia colores en `:root` dentro del `<style>`.
- Reemplaza `assets/hero.png` por otro personaje (misma proporción ~64x64).
- Ajusta la dificultad cambiando `gap`, velocidad (2.4) o gravedad/salto del héroe en JS.
